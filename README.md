# Flutter Hello World Application

A simple Flutter application demonstrating the creation of 
a basic Flutter project that displays "Hello World" on the screen. 
This project was built as part of Lab 2 for the Dart Programming course.

---

## Features

- Displays "Hello World" text on the screen.
- Can run on multiple platforms:
  - Android devices
  - Windows desktop
  - Web (via Chrome)
- Basic project structure suitable for Flutter beginners.

---

## Prerequisites

Before running this project, make sure you have:

- Flutter SDK installed (tested on Flutter 3.29.0)
- Android Studio or VS Code with Flutter & Dart plugins
- A physical Android device or emulator
- Optional: Chrome for web development, Visual Studio for Windows development
